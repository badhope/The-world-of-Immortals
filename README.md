## 🎯 What is The World of Immortals?

**The World of Immortals** is an immersive cultivation RPG game where players embark on a journey of self-improvement, martial arts mastery, and spiritual enlightenment.

> *"In the path of cultivation, every step is a battle against heaven and self."*

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🗡️ **Martial Arts Combat** | Strategic turn-based combat system |
| 🏔️ **Vast World** | Explore mountains, forests, and mystical realms |
| 🧘 **Cultivation System** | Progress through realms of power |
| 📜 **Ancient Secrets** | Discover hidden techniques and scrolls |
| 🎭 **Rich Story** | Engage with characters from Chinese mythology |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/badhope/The-world-of-Immortals.git

# Install dependencies
npm install

# Start the game
npm run dev
```
