# 修仙世界 - 项目计划书

## 项目概述

**项目名称**: 修仙世界 (The World of Immortals)  
**项目类型**: 桌面端修仙模拟游戏  
**技术栈**: Electron + Vue 3 + TypeScript + Pinia + SQLite  
**开发状态**: 开发中

## 核心设计理念

### 游戏风格
- **界面风格**: 古典仙侠风 - 深色背景，金色/红色点缀，古风元素
- **配色方案**: 深色金红 - 深色背景(#0a0a0f)，金色(#d4af37)和红色(#c04040)作为主色调
- **字体风格**: 混合字体 - 标题使用楷体(Noto Serif SC)，正文使用现代字体(Noto Sans SC)

### 布局设计
- **主布局**: 三栏式布局 - 左侧导航栏(200px) + 中间主内容区 + 右侧信息面板(240px)
- **导航方式**: 侧边导航栏 - 垂直导航，点击切换页面
- **信息密度**: 平衡模式 - 显示大部分信息，布局合理

## 核心系统设计

### 1. 事件系统
- **事件池**: 28+预设事件，包含故事、发现、危险、机遇、修炼等类型
- **触发条件**: 基于地点、境界、天数、物品、随机概率
- **事件选择**: 多选项分支，每个选择有不同成功率
- **后果系统**: 经验、生命、灵力、物品、解锁地点等效果
- **事件冷却**: 防止重复触发，记录已触发事件

### 2. 地图系统
- **地点数量**: 30+个地点 - 村庄、城池、山脉、森林、洞府、遗迹、秘境等
- **地点连接**: 完整的旅行网络，每个地点有旅行时间
- **危险系统**: 每个地点有不同的敌人和危险等级
- **资源系统**: 可采集资源，带有丰富度和再生时间
- **探索奖励**: 基于地点等级和玩家境界的奖励计算

### 3. 世界状态系统
- **时间推进**: 清晨→上午→中午→下午→傍晚→夜晚→深夜
- **季节变化**: 春夏秋冬循环（每30天）
- **天气系统**: 根据季节随机变化（晴朗、多云、小雨、大雨、雷暴、大雾、暴雪）
- **月相系统**: 8种月相，影响灵气浓度
- **灵气密度**: 综合天气、月相、时间、季节计算
- **修炼加成**: 基于世界状态的修炼效率加成

### 4. 战斗系统
- **战斗方式**: 回合制战斗 - 可以选择攻击、防御、技能等操作
- **敌人类型**: 野兽、散修、妖魔等
- **战斗奖励**: 击败敌人获得经验和物品

### 5. 合成系统
- **合成方式**: 配方列表式 - 显示所有配方，未解锁的显示为问号
- **合成类型**: 丹药炼制、装备锻造、物品合成
- **材料需求**: 显示所需材料和数量

### 6. 建筑系统
- **地点建筑**: 在当前地点建造功能性建筑
- **个人洞府**: 建造自己的洞府，可以升级和装饰
- **资源设施**: 建造资源生产设施，定期产出资源
- **修炼设施**: 建造修炼设施，提供修炼加成

### 7. 社交系统
- **NPC好感度**: 与NPC建立关系，影响剧情和交易
- **宗门系统**: 加入宗门，获得宗门任务和资源
- **师徒系统**: 收徒或拜师，获得指导和资源
- **修仙者互动**: 与其他修仙者竞争或合作

### 8. 存档系统
- **存档方式**: 混合模式 - 自动保存 + 手动存档位
- **存档数量**: 支持多个存档位
- **存档内容**: 玩家数据、世界状态、建筑、事件记录

## AI集成设计

### AI功能
- **事件文本生成**: AI生成事件描述和对话文本
- **NPC交互**: AI控制NPC行为和对话
- **世界扩展**: AI生成新的地点和故事
- **修炼指导**: AI提供修炼建议和策略

### AI支持
- **云端模型**: 支持OpenAI等云端AI服务
- **本地模型**: 支持Ollama等本地AI模型
- **可选功能**: AI为可选增强，游戏可在无AI情况下完整运行

## 界面设计详情

### 主界面（首页）
- **当前位置描述**: 显示当前地点的描述和可进行的活动
- **玩家状态面板**: 显示玩家的生命、灵力、境界等状态
- **快速行动按钮**: 显示快速操作按钮如采集、修炼、探索
- **事件日志**: 显示最近发生的事件和消息

### 探索页面
- **地点列表**: 显示所有已发现地点的列表，点击查看详情
- **地图视图**: 用图形化方式显示地点之间的连接关系
- **当前地点详情**: 显示当前地点的详细信息和可采集资源
- **NPC交互**: 显示当前地点的NPC和可交互对象

### 背包界面
- **展示方式**: 分类标签式 - 按类型分类显示
- **分类类型**: 材料、丹药、装备、任务物品等
- **物品详情**: 点击物品显示详细信息

### 修炼页面
- **境界进度**: 显示当前境界、经验进度、下一境界要求
- **功法系统**: 显示可学习的功法和已学功法
- **属性分配**: 显示属性点和可分配点数
- **修炼加成显示**: 显示修炼加成和世界状态影响

### 炼制页面
- **配方列表**: 显示所有配方，未解锁的显示为问号
- **材料检查**: 自动检查是否有足够材料
- **一键合成**: 选择配方后自动使用材料合成

### 洞府页面
- **建筑列表**: 显示可建造的建筑
- **已建建筑**: 显示已建造的建筑和状态
- **建筑升级**: 显示建筑升级选项

## 动画效果

- **页面切换动画**: 页面切换时有淡入淡出效果
- **获得奖励动画**: 获得物品或经验时有提示动画
- **按钮交互反馈**: 按钮点击有反馈效果
- **进度动画**: 修炼或采集有进度动画

## 技术架构

### 前端
- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router
- **样式**: CSS Variables + Scoped CSS

### 后端
- **运行环境**: Electron
- **数据库**: SQLite (better-sqlite3)
- **文件系统**: Node.js fs模块

│   │   ├── game/         # 游戏内页面
